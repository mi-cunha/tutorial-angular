Conhecendo o Angular

O que é o Angular? É uma plataforma de desenvolvimento que utiliza Typescript para criar aplicações web do tipo SPA (Single Page Applications). É um framework orientado a objetos, baseado em componentes e declarativos. Foi criado e mantido pelo Google e é um dos frameworks mais conhecidos e usados para desenvolvimento web. Vantagens de usar o Angular:
  • Framework baseado em componentes para construir aplicações web escaláveis
  • Tem uma extensa coleção de bibliotecas com variedade de recursos como rotas, envio de formulários, requisições HTTP por Ajax e mais. Tudo que é necessário para iniciar uma aplicação.
  • No Angular você pode testar seu código durante seu desenvolvimento.
  • Facilmente escalável: de um projeto individual a projetos de grandes corporações
