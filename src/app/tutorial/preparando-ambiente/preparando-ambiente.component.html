Preparação do Ambiente de Desenvolvimento

Ao criar um novo projeto, o Angular cria todos os arquivos iniciais necessários e também instala todas as dependências necessárias para seu funcionamento.

Faça a instalação da ultima versão do node.js para ter acesso ao comando npm

Abra seu terminal de preferência e digite: npm install -g @angular/cli
install- instala um novo pacote
g – instala globalmente, não sendo necessário instalar novamente em cada novo projeto criado

Como falado anteriormente em Conceitos, usamos alguns comandos específicos no Angular CLI:

  • devemos sempre digitar ng antes de todo comando que queremos executar
  • ng version para testar se o CLI foi corretamente instalado
  • ng new nome-da-aplicacao após entrar na pasta onde deseja que ele seja instalado

Durante a criação da aplicação no cmd, você pode escolher suas preferências, mas caso não esteja familiarizado com o Angular, sugiro as seguintes opções:

  • Escolher não ao criar as rotas
  • Optar por CSS como folha de estilo

Ao finalizar entre na pasta da aplicação recém criada: cd nome-da-aplicação

E em seguida execute o seguinte comando: ng serve Este comando executa a aplicação localmente como ambiente de desenvolvimento onde podemos alterar o código e ver o resultado em tempo real. obs.: na primeira vez ao executar o comando será perguntado se gostaria de compartilhar dados de uso com os desenvolvedores, fica a seu critério.

Após a mensagem de sucesso, abra a aplicação em seu navegador na porta indicada.

A porta padrão é localhost:4200

Para interromper o servidor de desenvolvimento a qualquer momento, utilize o atalho: ctrl + C
